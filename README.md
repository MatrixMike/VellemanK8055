
VellemanK8055
=============

Install
-------

* cabal update
* cabal install VellemanK8055



Linux
-----

lsusb
Bus 002 Device 008: ID 10cf:5500 Velleman Components, Inc. 8055 Experiment Interface Board (address=0)



Tests
-----

* cabal install --enable-tests VellemanK8055
* cabal build
* cabal test
